# news_site
News site project on javascript, react, vite

## Deployed site on github pages
[Gihub-Pages](https://grisha765.github.io/news_site/)

## Self-deployment
- **Install npm libs**:
    ```bash
    cd web/ &&\
    npm install
    ```

- **Run vite server**:
    ```bash
    npm run dev
    ```
